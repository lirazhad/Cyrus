# Cyrus

Getting stackoverflow question by user id and sort them by date, number of answers and number of views.
1. clone project
```
git clone https://github.com/lirazhad/Cyrus.git
```
2. Install packages  
```
$ yarn 
```
3. install pods  
```
$ cd ios && pod install
```

* iOS  
```
$ yarn ios
```
* Android  
```
$ yarn android
```
